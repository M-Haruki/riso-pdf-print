# 全体の流れ

## 必要なもの

- Windows PC(管理者権限が必要)
- USBメモリ(中身を消去してもいいもの)
- リソグラフ
  - SF625Ⅱ(生徒会室の古い新しい方)
  - SD5430(生徒会室の古い方)
  - その他の機種も同様の方法で印刷できる可能性がある
