# 全体の流れ

## 必要なもの

- Windows PC(管理者権限が必要)
- USBメモリ(中身を消去してもいいもの)
- リソグラフ
  - SF625Ⅱ
  - SD5430

リソグラフについては、他の機種でも同様の方法で印刷できる場合がほとんどです。
